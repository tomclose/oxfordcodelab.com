What will I learn?

The course will run over two terms. In this time you will learn how to put together a basic dynamic website (like Facebook, but simpler!).

In the first term you will focus mainly on static webpages. With no previous knowledge required, you will learn the basics of how the internet works. You will learn how to code your own personal webpage using HTML and CSS, and host it online for free using github pages. You will find out how to use git to manage your code as you develop and to collaborate with others - both essential skills whether you intend to go into startups, industry or academic research.

The second term will take your knowledge from the first term and extend it to buildinging dynamic sites - those that react to information provided by the user (most sites you use in daily life will be dynamic sites). You will learn the ruby programming language and, in particular, its applications to web development. You will learn how to use a database to store user information and use this to construct user-specific pages (like a personal twitter feed, or facebook news feed).

Will this course make me into a web developer?

Learning any new skill takes a lot of time and effort - after two terms you will certainly know some web development, but there will still be a lot to learn. The aim of this course is to give you the foundations to be able to teach yourself the rest.

How much does it cost?

The course is completely free for Oxford University students.

In the first few weeks you will be required to buy a domain name, as part of one of the exercises. This will cost a couple of pounds and is the only thing you will be required to buy.

When is it?

The course will run Tuesdays 6-7.30pm from 2nd - 8th week.

How much work will it be?

The course will require at least 4 hours work outside class each week. Based on past experience, if you don't put this time in you won't get very little out of the course.

How will you select people?

Places on the course are limited. We will aim to select those who will get the most out of the course.

Is it just for girls?

No. There will also be a Code First: Girls course running this term though - if you are female, you might want to apply for that too!
